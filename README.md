# NFC Cards
So, I’ve been messing around with NFC tech lately. This repo is basically the home for my NFC business card project. I wanted to make something cooler than a piece of paper, so I designed a custom PCB that doubles as a digital business card using the NT3H1101 chip.

## Specifications

- **IC** - NT3H1101
- **Frequency** - 13.56 MHz

## How to make your own

- Clone the repository (or download the kicad folder). Open it. Modify the silkscreen to your likings. I used Canva. Then export gerber files. And Get it manufactured (JLCPCB.PCBWAY,etc)
- Dont forget to post yours online!
